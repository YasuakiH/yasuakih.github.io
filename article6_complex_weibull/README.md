# 複雑なワイブル — 混合・競合故障モードのシミュレーション

> **Note:** このページは信頼性工学の学習記録であり、CC0-1.0 ライセンスでパブリックドメインに公開している。

---

## 背景と目的

ワイブル確率プロット（ワイブルプロット）は信頼性解析の基本ツールだが、
現実の製品データでは打点が直線に乗らないことがある。
その原因として代表的なのが次の 2 つである。

| 現象 | 原因 |
|------|------|
| プロットが S 字やカーブを描く | **競合故障モード**（バスタブ曲線） |
| プロットが折れ線状になる | **混合母集団**（良品・不良品の混在） |

本ページでは、これら 2 種の現象をスライダーで操作しながら視覚的に理解できる
インタラクティブシミュレーターを Python で実装した。

---

## デモ 1 — 競合故障モード（バスタブ曲線）

**スクリプト:** [`competing_risks.py`](./competing_risks.py)

![competing_risks.png](img/competing_risks.png)

### モデルの構造

製品寿命を 3 種のワイブル分布の競合として表現する。

| 故障モード | α | β | 特徴 |
|-----------|---|---|------|
| 初期故障  | 50  | 0.5 | β < 1：ハザードが時間とともに**減少** |
| 偶発故障  | 200 | 1.0 | β = 1：ハザードが**一定**（指数分布） |
| 摩耗故障  | 500 | 3.5 | β > 1：ハザードが時間とともに**増大** |

偶発故障をベースに、初期故障と摩耗故障のウェイトをスライダーで調整することで、
任意のバスタブ形状を作り出せる。

### 操作手順

1. **初期状態**（スライダー両方 0）: 偶発故障のみ → ワイブル打点はほぼ直線
2. **初期故障ウェイトを増やす** → 初期ハザード上昇、打点の前半部が右に曲がり傾き m が緩くなる
3. **摩耗故障ウェイトを増やす** → ハザードが右上がり、打点の後半部が左に曲がり傾き m が急になる
4. **両方を増やす** → バスタブ型ハザード、ワイブルプロットが S 字を描く

### 実装のポイント

混合ハザードは各群の PDF と生存関数の加重和から導出する。

```
h_mix(t) = f_mix(t) / S_mix(t)
         = Σ wᵢ fᵢ(t) / Σ wᵢ Sᵢ(t)
```

B10 ライフは累積ハザード H(t) = ∫h(t)dt を数値積分し、F(t) = 1 − exp(−H(t)) が
0.10 に最も近い時刻として推定する。

## デモ 2 — 混合母集団（良品・不良品の混在）

**スクリプト:** [`mixture.py`](./mixture.py)

![mixture.png](img/mixture.png)

### モデルの構造

「良品」と「不良品」が異なるワイブル分布を持ちながら混在する母集団を想定する。

| 群 | デフォルト B10 | デフォルト β | 特徴 |
|----|--------------|------------|------|
| 良品 | 1200 | 2.0 | 摩耗故障型 |
| 不良品 | 600 | 0.8 | 初期故障型 |

### 操作手順

1. **初期状態**（Weight Bad = 0）: 良品のみ → ワイブルプロットはほぼ直線
2. **Weight Bad を 1% ずつ増加** → 不良品が混入、プロットが直線からずれ始める
3. 各スライダーで良品・不良品の B10 ライフや β を変えて挙動を観察する

### 実装のポイント

混合 PDF は厳密な加重和として計算する。

```
f_mix(t)  = w_good · f_good(t) + w_bad · f_bad(t)
h_approx(t) ≈ w_good · h_good(t) + w_bad · h_bad(t)  （近似）
```

混合ハザードの加重平均は近似だが、教育目的での直感的理解には十分である。
厳密には `f_mix / S_mix` で計算すべき点に注意。

## 学習のまとめ

| 観察 | 示唆される解釈 |
|------|--------------|
| ワイブルプロットが直線に乗る | 単一のワイブル分布が妥当 |
| 前半が右に曲がる（傾き↓） | 初期故障成分が混在 |
| 後半が左に曲がる（傾き↑） | 摩耗故障成分が混在 |
| S 字を描く | 複数の故障モードが競合 |
| 折れ線状になる | 良品・不良品などの混合母集団 |

単一のワイブル分布でフィットしようとすると、誤った β を推定し
設計寿命や保全計画を誤るリスクがある。プロットの形状から母集団構造を
仮説として立て、混合モデルや競合リスクモデルを検討することが重要である。
