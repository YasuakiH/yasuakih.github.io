# 【制作中】モンテカルロ法によるデジタル印刷機の顧客隠れパラメータ推定

この記事は、デジタル印刷機を用いてオンデマンド印刷を営む印刷業者が製造している印刷物の特徴を、コンピュータシミュレーションによって推定するものである。

## 導入
オンデマンド印刷を営む印刷業などで用いられるデジタル印刷機は、その性能を維持するために、ドラム、フィルタ、ブレード、ベルトなどの消耗品 (定期交換部品とよぶ) を定期的に交換する必要がある。メーカーでは、定期交換部品ごとに寿命の管理目標を設定し、計画的に交換することで印刷機の故障を未然に防ぐ、保守サービス (予防保守) を提供する。高い信頼性が求められる印刷機の場合、通信ネットワークを介して稼働状況を送信する機能を備え、遠隔監視を可能としている。メーカーも印刷業者 (顧客) から提供された情報を利用して遠隔保守をすることができる。顧客情報保護の観点から、その情報は保守に必要な機械の稼働状況に限られ、どのように機械を使うかなど顧客固有の情報は含まない。そこで、予防保守を顧客の業務に合わせて最適化するために、収集可能な情報を基にして、外部から観察できない顧客の使い方を特徴付ける「隠れパラメータ」を推定することが考えられる。こうした隠れパラメータには、物理的な用紙サイズ・印刷ページ数・両面片面の別、原稿を特徴づけるインク塗布量、また印刷機が生産財を出力する点では顧客固有の品質基準などがあげられる。モンテカルロ法は、このような隠れパラメータを推定するコンピューターの応用である。

## 目的
デジタル印刷機の保守サービスを最適化するプロセスをコンピュータシミュレーションで実現する。全体を3つの独立したテーマに分割して、この記事はその最初のステップを説明する。架空の印刷機を想定し、その隠れパラメータを推定するシミュレーションを、汎用のプログラミング言語Pythonとフリーソフトで構築する。

* 1. 顧客の隠れパラメータ推定 【本記事の範囲】
* 2. 部品ライフ推定
* 3. 機械の信頼度成長

## 背景
### 印刷ジョブ
印刷業者が1つ1の印刷物を製造する工程を印刷ジョブと呼び、次のようなパラメータの集まりである。通常これらは印刷業者のビジネスの成績にもつながる点において秘匿性の高い情報である。

* トータルエリアカバレッジ: 用紙面積に対するインクが塗布された面積の割合 (TAC値とも)
* 印刷ページ長: 印刷ジョブに含まれるページ数。複数部数の場合は印刷部数だけ同じ内容が繰り返される。
* 用紙サイズ: 印刷ジョブで指定される用紙の大きさ (例: はがき、A5、A4、B4、A3ノビ、長尺)
* 両面比率: 印刷物全体に対する両面印刷の比率。(両面印刷/(両面印刷+片面印刷))

カラー印刷物の場合は、色に関するパラメータも必要であるが、この記事では簡略化のためにモノクロとした。(カラー比率は0%)

* カラー比率: 印刷物全体に対するカラー印刷の比率。(カラー印刷/(カラー印刷+モノクロ印刷))

### 遠隔保守で収集される情報
印刷機から遠隔保守のために送信される情報は、印刷機が消費したインクや用紙の量である。遠隔保守では、これらに基づき消耗品の消耗度合いを推定し、次の交換時期を推定する。

* インク総量: 印刷機が消費したインクの量
* 用紙サイズ分布: 印刷ジョブに含まれる用紙サイズの分布
* 印刷ページ長分布: 印刷ジョブに含まれる印刷ページ長の分布

### 定期交換部品と部品ライフ
印刷機の性能を維持するために定期的に交換される部品を定期交換部品という。これらの部品は使用によって消耗するので、一定の基準に基づいて交換を要する。部品の消耗度合いを部品ライフといい、例えばその部品で印刷した用紙の枚数数や、部品の運動した回数や距離のような指標で測定される。メーカーは、機械の設計時に部品ライフの管理目標を策定し、保守サービスではこれを基準に部品を交換する。

### 予防交換と故障交換
定期交換部品の交換では印刷機を止める必要があるため、印刷業者の業務計画に沿うことが望ましい。複数の保守作業を同時にするなど計画的にすることで機械の稼働率を維持できる。計画的な保守では、部品ライフの推定値が管理目標に到達する前であっても交換する「予防交換」が行われる。これに対して突発的な故障が起こって稼働中の機械が止まった場合は、回復が急がれるため、問題を起こした部品のみ交換する「故障交換」とよばれる保守形態がなされる。
