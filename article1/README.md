# モンテカルロ法によるデジタル印刷機の顧客隠れパラメータ推定【制作中】

## 導入
この記事は、デジタル印刷機を用いてオンデマンド印刷を営む印刷業者が、どのような印刷物を製造しているのかを、コンピュータシミュレーションで推定しようとするものである。デジタル印刷機はその性能を維持するため、ドラム、フィルタ、ブレード、ベルトなど定期交換部品 (消耗品) を定期的に交換する必要がある。メーカーでは、定期交換部品ごとに寿命の管理目標を設定し、計画的に交換する予防保守をすることで印刷機の故障を未然に防ぎつつ、保守サービスの提供による収益性を確保している。高い信頼性が求められる印刷機では、遠隔監視をするために稼働状況を通信ネットワークを介してメーカーに送信する機能を備えている。しかし、顧客情報保護の観点から収集する情報は最低限とされ、高度な予防保守に必要とされる
詳細は得られない。そこで収集可能な情報を元に、外部から観察できない「隠れパラメータ」を推定して代用する必要がある。隠れパラメータとして、印刷物の特徴には物理的な用紙サイズ・印刷ページ数・両面片面の別、また原稿の特徴にはインク塗布量、また印刷機が生産財を出力する点では顧客に固有の品質基準も含まれる。この記事では架空の印刷機を想定して、モンテカルロ法により隠れパラメータを推定する。Pythonプログラミング言語とフリーソフトで構築する。

このシミュレーションでは、デジタル印刷機の保守サービスを最適化する一連のプロセスでの使用を想定した。

### 隠れパラメータの推定
シミュレーションで必要とするのは、印刷業者が製造する印刷物がどのような要因、量、種類で構成されるか (印刷ジョブという) である。

* トータルエリアカバレッジ: 用紙面積に対するインクが塗布された面積の割合 (TAC値とも)
* 印刷ページ長: 印刷ジョブに含まれるページ数。複数部数の場合は印刷部数だけ同じ内容が繰り返される。
* 用紙サイズ: 印刷ジョブで指定される用紙の大きさ (例: はがき、A5、A4、B4、A3ノビ、長尺)
* 両面比率: 印刷物全体に対する両面印刷の比率。(両面印刷/(両面印刷+片面印刷))

カラー印刷物の場合は次のパラメータも必要となるが、ここでは簡略化のためにモノクロとした。(カラー比率は0%)

* カラー比率: 印刷物全体に対するカラー印刷の比率。(カラー印刷/(カラー印刷+モノクロ印刷))

### 利用できる情報
印刷機から遠隔保守のために送信される情報は次のようなものがある。課金や保守のために集計されていて、部品ライフを予測するための要求を満たせない。

* インク総量: 印刷機が消費したインクの量
* 用紙サイズ分布: 印刷ジョブに含まれる用紙サイズの分布
* 印刷ページ長分布: 印刷ジョブに含まれる印刷ページ長の分布
