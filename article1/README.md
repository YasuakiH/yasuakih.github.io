# モンテカルロ法によるデジタル印刷機の顧客隠れパラメータ推定
* * *
## 編集中
* * *

## 概要
デジタル印刷機を用いてオンデマンド印刷を営む印刷業者が、どのような印刷物を印刷しているのかコンピュータシミュレーションで推定する。印刷物の特徴には、用紙サイズ、印刷ページ数、両面片面の別、インクの塗布量など多くがある。印刷機メーカーでは、印刷機で使われる定期交換部品 (印刷ドラムなど) の寿命の管理目標を設定し、予防保守をすることで印刷機の故障を未然に防ぎつつ、部品をできるだけ長く使うことで収益性を改善することができる。

高信頼性が求められる印刷機では、遠隔保守を目的として運転状況を定期的に記録し、インターネット介して印刷機メーカーに送信する機能を持つ。ただし顧客情報保護の観点から収集する情報は限られ、前述のような情報の大半は得られない。そこで収集可能な情報を元に、その裏に隠された情報を推定することが行われる。

## 要求
シミュレーションで推定したいのは、業者が受注する注文がどのような「印刷物種類」で構成されるかの比率である。

### 印刷物種類:
はがき、名刺、ビジネス文書、書籍、パンフレット、チラシ、カタログ、ポスターのような一般的な名称である。

この問題をシミュレーションで解くために、潜在的なパラメータを特定したモデルを作成し、それぞれの印刷物種類を特徴付ける「潜在的なパラメータ」も推定したい。

### 潜在的なパラメータの例を示す。
* 印刷物種類: はがき、名刺、ビジネス文書、書籍、パンフレット、チラシ、カタログ、ポスターのような一般的な名称である。
* TAC値: 印刷物の種類によって統計的分布を仮定する。
* 用紙サイズ: はがき、A4、B4、A3、長尺。
* ページ数: 印刷物種類の1部あたりのページ数である。範囲は正規分布を仮定する。
* 部数: 1つの注文に含まれる印刷物種類の部数である。総ページ数は、ページ数 x 部数 x 両面比率で求められる。
* 両面比率: 印刷物全体に対する両面印刷の比率。(すなわち両面印刷/(両面印刷+片面印刷))
* カラー比率: 印刷物全体に対するカラー印刷の比率。(すなわちカラー印刷/(カラー印刷+モノクロ印刷))
* 受注比率: 注文全体に対する、その印刷物種類の比率。たとえば、ビジネス文書の注文率 = ビジネス文書の注文数/全体の注文数とする。

### 印刷物種類ごとの、ページ数、および部数をそれぞれ説明したい。

このほかに追加を検討したいパラメータ。

1. 注文頻度（注文の頻繁さ）
2. カラー印刷 vs モノクロ印刷の割合
3. 納期
4. 顧客タイプ（法人 vs 個人）
5. 印刷物の仕上げ方法（仕上げ加工の有無）
6. 注文のカスタマイズ度合い（標準 vs カスタマイズ）
