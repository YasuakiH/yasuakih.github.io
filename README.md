# 定期交換部品の交換時期の最適化に関するPython言語によるシミュレーション

ビジネス、特に修理可能な製品をもつメーカーでは、顧客先に保守作業員を派遣し、消耗品 (定期交換部品) を定期的に交換するなど、製品の安定稼働と性能維持のための保守サービスを提供している。こうした保守サービスを計画的に運用するためには、消耗品が所定の寿命を果たすことが前提となる。もし、寿命が計画よりも短ければ製品は予期せず故障し、顧客は業務に影響を受けることになる。一方メーカーも、保守作業員を緊急に派遣して故障を修復し、機能を回復しなければならない。そこで製品には、計画外の保守発生を防ぎ、顧客業務の停止を避けるために、稼働状況をメーカーに定期的に通知する仕組みが備わっている。

定期交換部品の交換時期を決めるためには寿命を推定する必要がある。しかしその寿命には、顧客の用途や品質基準などさまざまな要因 (隠れパラメータ) が影響するため、メーカーでは把握できないことが多い。この記事では、(1) 判明している情報から顧客の「隠れパラメータ」を推定し、(2) それに基づいて定期交換部品の寿命を推定し、(3) 最適な交換時期を決めるというテーマを取り扱う。例題として架空の印刷機を考え、各テーマについてコンピュータシミュレーションを設計し、Python言語によって実装する。
