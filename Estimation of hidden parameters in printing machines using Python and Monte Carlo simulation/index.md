# Pythonとモンテカルロシミュレーションを用いた印刷機の運用状況の推定

目的: デジタル印刷機を用いてオンデマンド印刷を営む業者が、どのような注文を得意とするのかコンピュータシミュレーションで推定する。

シミュレーションで推定したいのは、業者が受注する注文がどのような「印刷物種類」で構成されるかの比率である。
印刷物種類: はがき、名刺、ビジネス文書、書籍、パンフレット、チラシ、カタログ、ポスターのような一般的な名称である。

この問題をシミュレーションで解くために、潜在的なパラメータを特定したモデルを作成し、
それぞれの印刷物種類を特徴付ける「潜在的なパラメータ」も推定したい。

潜在的なパラメータの例を示す。
* 印刷物種類: はがき、名刺、ビジネス文書、書籍、パンフレット、チラシ、カタログ、ポスターのような一般的な名称である。
* TAC値: 印刷物の種類によって統計的分布を仮定する。
* 用紙サイズ: はがき、A4、B4、A3、長尺。
* ページ数: 印刷物種類の1部あたりのページ数である。範囲は正規分布を仮定する。
* 部数: 1つの注文に含まれる印刷物種類の部数である。総ページ数は、ページ数 x 部数 x 両面比率で求められる。
* 両面比率: 印刷物全体に対する両面印刷の比率。(すなわち両面印刷/(両面印刷+片面印刷))
* カラー比率: 印刷物全体に対するカラー印刷の比率。(すなわちカラー印刷/(カラー印刷+モノクロ印刷))
* 受注比率: 注文全体に対する、その印刷物種類の比率。たとえば、ビジネス文書の注文率 = ビジネス文書の注文数/全体の注文数とする。

印刷物種類ごとの、ページ数、および部数をそれぞれ説明したい。

このほかに追加を検討したいパラメータ。

1. 注文頻度（注文の頻繁さ）
2. カラー印刷 vs モノクロ印刷の割合
3. 納期
4. 顧客タイプ（法人 vs 個人）
5. 印刷物の仕上げ方法（仕上げ加工の有無）
6. 注文のカスタマイズ度合い（標準 vs カスタマイズ）
